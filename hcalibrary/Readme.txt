# Project Details

Name: hcalibrary

Author: Umesh Agrawal

This code is deployed as a Azure Web App.

Steps. 

1. Download the project.
2. Open folder in hcalibrary folder in visual studio code
3. Run `npm start` command in the terminal in the folder path

Above command Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
